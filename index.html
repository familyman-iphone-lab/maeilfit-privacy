개인정보처리방침

매일핏 (MaeilFit) · 최종 수정일: 2025년 1월 1일

한 줄 요약

매일핏은 서버가 없는 완전한 로컬 앱입니다.
모든 운동 데이터는 오직 내 기기에만 저장되며, 외부로 전송되지 않습니다.

광고 없음 · 추적 없음 · 분석 SDK 없음 · 개인정보 수집 없음

1. 수집하는 개인정보

매일핏은 개인을 식별할 수 있는 정보를 수집하지 않습니다.

앱이 기기에 저장하는 데이터는 아래와 같으며, 모두 사용자 본인의 기기에만 보관됩니다.

데이터 종류저장 위치목적
운동 기록
(세션, 세트, 무게, 횟수)		 기기 내부
(SwiftData)		 운동 기록 저장 및 통계 제공
앱 첫 실행일 기기 내부
(UserDefaults)		 무료 체험 기간 계산
화면 모드 설정 기기 내부
(UserDefaults)		 다크/라이트 모드 유지
휴식 타이머 기본 시간 기기 내부
(UserDefaults)		 사용자 설정 유지

2. 인앱 결제

프리미엄 기능 구매는 Apple App Store를 통해서만 처리됩니다.

결제 정보(카드 번호, Apple ID 등)는 Apple이 관리하며, 매일핏 개발자는 결제 정보에 접근할 수 없습니다.

Apple의 개인정보 처리방침: apple.com/kr/legal/privacy

3. 푸시 알림

매일핏은 로컬 알림만 사용합니다. (휴식 타이머 종료 알림)

  • 알림은 외부 서버 없이 기기 자체에서 발송됩니다.
  • 알림 관련 정보는 외부로 전송되지 않습니다.
  • iOS 설정 앱에서 언제든지 알림을 끌 수 있습니다.

4. 데이터 내보내기 · 복구

설정에서 운동 기록을 JSON 파일로 내보낼 수 있습니다.

  • 내보낸 파일은 사용자가 직접 선택한 위치에만 저장됩니다.
  • 파일은 기기의 임시 폴더에 생성된 후 iOS가 자동으로 정리합니다.
  • 내보낸 파일을 어디에 저장하고 공유할지는 전적으로 사용자가 결정합니다.

5. 제3자 제공 및 외부 전송

매일핏은 사용자 데이터를 제3자에게 제공하거나 외부 서버로 전송하지 않습니다.

  • 광고 네트워크 없음
  • 분석 서비스(Firebase, Amplitude 등) 없음
  • 소셜 로그인 없음
  • 외부 API 호출 없음

6. 사용자 권리 및 데이터 삭제

사용자는 언제든지 자신의 데이터를 완전히 삭제할 수 있습니다.

  • 앱 삭제: iOS 기기에서 앱을 삭제하면 모든 운동 데이터가 즉시 삭제됩니다.
  • 개별 기록 삭제: 앱 내에서 특정 세션이나 운동을 개별 삭제할 수 있습니다.
  • 데이터 이동: JSON 내보내기를 통해 언제든지 데이터를 다른 기기나 앱으로 이동할 수 있습니다.

7. 아동 개인정보 보호

매일핏은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

개인정보를 수집하지 않는 로컬 전용 앱이므로 연령 제한 없이 안전하게 사용할 수 있습니다.

8. 개인정보처리방침 변경

본 방침은 법령 변경이나 앱 기능 추가 시 업데이트될 수 있습니다.

변경 시 앱 업데이트 노트 또는 본 페이지를 통해 안내드립니다.
